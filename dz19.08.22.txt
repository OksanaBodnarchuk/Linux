   1. Linux это семейство открыоперационных систем на базе ядра Linux с набором свободного и открытого програмного обеспечения. Чаще используется програмными администраторами для удаленного доступа к серверам и базам данных.

2. CLI (Command Line Interface) интерфейс командной строки (терминал и пр.), GUI (Grafphical User Interface) графический интерфейс пользователя (привычный вид ОС для обычных пользователей).

3. SSH (Secure Shell) сетевой протокол (подключение к серверу).

4. Веести в строке комбинацию "ssh-keygen -t rsa" и нажать Enter. Это нужно для того, чтобы входить на известный нам сервер без паролья.

5.Набрать в строке "ssh-copy-id <SERVER>"

6. С помощью команды "gzip" происходит архивация (сжатие) файла.

7. Перенаправление делается с помощью  команды > или >>.
