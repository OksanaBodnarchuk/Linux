Git — это система, которая позволяет сразу нескольким разработчикам сохранять и отслеживать изменения в файлах вашего проекта.

Репозиторий — это все файлы, находящиеся под контролем версий, вместе с историей их изменения и другой служебной информацией.

.gitignore нужен для скрытия ненужных для коммита файлов и папок

Git-commit — это операция, которая берет все подготовленные изменения и отправляет их в репозиторий как единое целое.

Настройка аккаунта локального пользователя:
git config --global user.name 'name urname'
git config --global user.email 'root@ab88.pro'
git config --global core.editor 'core --wait' (привязка к VScode)

Создание репозитория:
Зайти на сайт GitHub в свой аккаунт, в верхнем правом углу нажать + со стрелочкой, выбрать new repository. в поле написать название и нажать зеленую кнопку Create repository.

Добавление файлов в индекс и выполнение коммита:
git init (создать новый реп)
git add . (добавить в очередь на коммит все файлы)
git commit -m 'Любой комментарий к комиту'

Просмотр журнала коммитов:
git status
